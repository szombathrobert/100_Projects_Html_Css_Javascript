Mi a JavaScript?
A JavaScript programozási nyelv egy objektumorientált, prototípus-alapú szkriptnyelv, amelyet weboldalakon elterjedten használnak. Ebből fejlődött ki a TypeScript, ami a JavaScript típusos változatának tekinthető
Java és JavaScript
Főbb különbségek a Java és a JavaScript között: A Java egy OOP programozási nyelv, míg a Java Script egy OOP szkriptnyelv. A Java olyan alkalmazásokat hoz létre, amelyek virtuális gépen vagy böngészőben futnak, míg a JavaScript kód csak böngészőben fut. A Java kódot le kell fordítani, miközben a JavaScript kód csak szövegben van.
Megtanulhatom a JavaScript-et Java nélkül?
Nem, nem szükséges ismerni a Java nyelvet a JavaScript megtanulása előtt. Bár ezeknek a programozási nyelveknek a nevei összefüggésre utalhatnak, valójában meglehetősen eltérőek a szintaxis, a cél és a használat tekintetében.
JavaScript & EcmaScript
A JavaScript alapnyelvét az ECMA TC39 bizottsága szabványosította ECMAScript néven. Az "ECMAScript" a nyelvi szabvány kifejezése, de az "ECMAScript" és a "JavaScript" felcserélhetően használhatók. Ezt az alapnyelvet nem böngészős környezetekben is használják, például a Node-ban.
